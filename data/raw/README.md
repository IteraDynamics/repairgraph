# Raw Data

This directory is reserved for customer-authorized source repair documentation used during local development and controlled ingestion.

Raw OEM repair documents should not be committed to this public repository unless their use and redistribution are explicitly authorized.

Recommended local-only organization:

```text
data/raw/honda/2025/crv/
  rear_side_outer_panel_replacement.pdf
  roof_and_side_panel_construction.pdf
```

RepairGraph should prioritize normalized structured outputs in `data/normalized/` over raw document storage.
